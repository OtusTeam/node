# Как мы можем запускать наше приложение?

Есть приложение, которое веб-сервер REST API - хотим его опубликовать, то есть чтобы оно было доступно в интернете
Будет домен, по которому можно стучаться и работать.

- Нам нужно определить как мы будем распространять наше приложение. 1 опция - сделать Dockerfile. Другие опции где-то билдить + как-то доставлять на хостинг.
- Написать CI/CD пайплайны. CD - доставка на хостинг
У меня на проектах - это Github Actions: билд докер образа -> публикация докер образа -> дальше через cloud говорим ему, что надо запустить новый образ.
Если делает "на коленке". Много делают руками и, как правило, копируют исходники(или пулят их на хостинге) и делают билд + запуск.
- Покупаем хостинг и домен - это VDS/CPU/Mem/Free Space/Traffic
- HTTPS - нужен


Для длительной поддержке нужен мониторинг.
Мы просто создаем агента, который проверяет сертификат
